# ChasingHorizons
追光而行：中国绝美自驾路线规划指南

《追光而行：中国绝美自驾路线规划指南》是一部专为国内自驾游爱好者、公路旅行家和人文地理探索者打造的保姆级路线指南。本书深度整合了中国全部 34 个省级行政区的经典自驾游路线，并结合了精准的地理位置数据、实际路段距离、可视化路线图，以及极具画面感的人文景观描述。

---

## 📖 书籍核心特色

1. **全景覆盖，丈量华夏**：全面覆盖中国 34 个省级行政区（包括 4 个直辖市、23 个省、5 个自治区、2 个特别行政区和台湾省）。
2. **严谨的地理验证**：所有自驾路线点位均经过省内地理围栏 bounding box 校验与地址组件二次校验，杜绝重名引发的跨省偏航。
3. **精准的里程路段**：各条路线均生成了详细的点对点里程表与总行程，结合水上公路、草原天路等各种特色路段。
4. **直观的路线地图**：利用 Google 静态瓦片地图，将各路线的起终点、中间途经点及行车轨迹绘制成精美的可视化图表，内置于每个章节中。
5. **极具审美的人文特征描述**：针对各线路量身定制了特点描述，饱含诗意与审美画面感，带您深度感悟祖国大好河山的山水神韵与人文信仰力量。

## 🚗 如何使用指南？

1. 进入 `docs/` 目录，挑选您感兴趣的省份指南 Markdown 文件。
2. 依据各线路的**自驾线路**和**路段距离表**合理规划日程，并参考配套的**可视化地图**做导航直观预判。
3. 结合**特点**和景点打卡推荐，了解每个落脚点的最佳体验项目，确保安全并收获极致的自驾体验。
