### 经典旅游路线与自驾路线

#### 路线一：环北部湾滨海椰风自驾线
* **特点**：看广西迷人的海滨风光，逛古港渔村，享火山岛探奇。
* **行车路线**：南宁 → 钦州三娘湾（看中华白海豚） → 防城港东兴金滩（中越边境） → 北海银滩/涠洲岛。

#### 路线二：中越边境山水画廊与桂林漓江经典自驾线
* **特点**：穿越德天瀑布的跨国山水画廊，泛舟桂林漓江水墨山水。
* **行车路线**：南宁 → 崇左（德天跨国瀑布/明仕田园） → 靖西（通灵大峡谷） → 那坡 → 百色（百色起义纪念馆） → 巴马（长寿之乡） → 河池（七百弄） → 柳州 → 桂林（漓江/阳朔西街/龙脊梯田）。
