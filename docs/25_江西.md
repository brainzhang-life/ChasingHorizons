### 经典旅游路线与自驾路线

#### 路线一：环江西红土水乡经典自驾大环线
* **特点**：融匡庐秀色、婺源水乡、千年瓷都、道教祖庭与红色摇篮为一体。
* **行车路线**：南昌（滕王阁/八一起义纪念馆） → 九江（庐山/鄱阳湖） → 景德镇古窑民俗博览区 → 婺源（江湾/篁岭晒秋） → 上饶三清山 → 鹰潭龙虎山 → 抚州文昌里 → 瑞金革命旧址 → 赣州宋代古城墙 → 吉安井冈山 → 萍乡武功山 → 新余仙女湖 → 宜春明月山 → 南昌。
