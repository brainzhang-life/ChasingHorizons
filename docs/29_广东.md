### 经典旅游路线与自驾路线

#### 路线一：粤北喀斯特山水生态自驾线
* **特点**：奇特的岭南喀斯特峰林，温泉、古刹与茶海森林。
* **行车路线**：广州 → 清远英德九龙峰林走廊 → 连州地下河 → 韶关（丹霞山/南华禅寺） → 乐昌九峰山 → 河源万绿湖 → 梅州（客家围龙屋/雁南飞茶田）。

#### 路线二：粤东粤西海丝风情黄金海岸线
* **特点**：沿着漫长的南海海岸线，享受阳光沙滩、海鲜饕餮与客家、潮汕人文。
* **行车路线**：湛江（东海岛/中国第一滩） → 茂名（放鸡岛） → 阳江（海陵岛大角湾） → 江门（开平碉楼与村落/下川岛） → 珠海（情侣路/外伶仃岛） → 深圳（大鹏半岛/大梅沙） → 惠州（巽寮湾/双月湾） → 汕尾（红海湾） → 汕头（南澳岛） → 潮州（广济桥/牌坊街）。
