## **广东经典自驾游线路推荐**

#### 广东喀斯特景观自驾游路线

* **自驾线路**：湟川三峡→连州地下河→连州→广东第一峰→广东峡天下→韶关丹霞山→英德宝晶宫→肇庆七星岩  
* **路线路段距离与地图**
  | 起点 | 终点 | 距离 |
  | :--- | :--- | :--- |
  | (1) 湟川三峡 | (2) 连州地下河 | 47.5 公里 |
  | (2) 连州地下河 | (3) 连州 | 35.2 公里 |
  | (3) 连州 | (4) 广东第一峰 | 93.2 公里 |
  | (4) 广东第一峰 | (5) 广东峡天下 | 412.7 公里 |
  | **总里程** | | **588.6 公里** |
  
  ![路线地图](images/maps/29_广东_01.png)
  
  
  
* **特点**：这是一条饱览岭南喀斯特地貌奇观、峰林与地下河的避暑生态自驾线。在连州地下河，您将乘小舟在巨大的溶洞地下暗河中行进，观赏五彩斑斓的钟乳石奇观，在湟川三峡看悬崖飞瀑飞泻入江；在英德九龙峰林走廊自驾，看连绵起伏的奇峰间散落着古朴村落与油菜花海；最终抵达肇庆七星岩，看湖中有山、山中有洞的岭南山水大观。

#### 珠三角及全省名胜自驾游路线

* **自驾线路**：广州市→河源市→汕头市→惠州市→深圳市→东莞市→中山市→珠海市→江门市→阳江市

  ![路线地图](images/maps/29_广东_02.png)
  
  
  
* **特点**：这是一条一网打尽岭南道教名山、蔚蓝海岛与温泉养生的广东全景自驾线。登上道教圣地惠州罗浮山，观万亩杜鹃红遍山野，在南昆山和万绿湖畔看秋日枫叶如火；驱车前往湛江东海岛、阳江海陵岛与汕头南澳岛，享受南海的阳光、沙滩与饕餮海鲜；最后在从化和珠海温泉小镇的氤氲泉水中，洗去一身尘埃，尽享度假慢生活。

#### 广东最美秋色自驾游路线

* **自驾线路**：广州市→石门国家森林公园→河源万绿湖→韶关丹霞山→帽子峰森林公园→坪田银杏森林公园→南岭国家森林公园  
* **路线路段距离与地图**
  | 起点 | 终点 | 距离 |
  | :--- | :--- | :--- |
  | (1) 广州天河体育中心 | (2) 石门国家森林公园 | 98.8 公里 |
  | (2) 石门国家森林公园 | (3) 河源万绿湖 | 105.2 公里 |
  | (3) 河源万绿湖 | (4) 南岭国家森林公园 | 288.7 公里 |
  | **总里程** | | **492.7 公里** |
  
  ![路线地图](images/maps/29_广东_03.png)
  
  
  
* **特点**：这是一条追逐粤北金秋红叶与南雄千年银杏的广东限定赏秋自驾景观路。从从化石门国家森林公园的漫山红叶出发，北上河源万绿湖畔，看万顷波光与红枫交相辉映；最精彩处在于自驾至韶关南雄帽子峰和坪田银杏林，深秋时节，千年古银杏树叶金黄落地，在泥墙黛瓦的客家大围前铺成金色地毯，宛如步入金色的童话世界。

#### 广东最美公路自驾游路线（S244最美旅游公路）

* **自驾线路**：广州市→新丰云髻山景区（图作“云医山”）→翁源县城→满堂客家大围→司前镇→深渡水瑶族乡→车八岭风景区→始兴县→铜钟寨→南雄帽子峰林场→坪田银杏森林公园→珠玑古巷→梅关古道  
* **路线路段距离与地图**
  | 起点 | 终点 | 距离 |
  | :--- | :--- | :--- |
  | (1) 广州天河体育中心 | (2) 翁源县城 | 190.1 公里 |
  | (2) 翁源县城 | (3) 满堂客家大围 | 60.7 公里 |
  | (3) 满堂客家大围 | (4) 司前镇 | 315.2 公里 |
  | (4) 司前镇 | (5) 深渡水瑶族乡 | 352.8 公里 |
  | (5) 深渡水瑶族乡 | (6) 始兴县 | 22.2 公里 |
  | (6) 始兴县 | (7) 铜钟寨 | 389.7 公里 |
  | (7) 铜钟寨 | (8) 珠玑古巷 | 428.8 公里 |
  | (8) 珠玑古巷 | (9) 梅关古道 | 18.9 公里 |
  | **总里程** | | **1778.4 公里** |
  
  ![路线地图](images/maps/29_广东_04.png)
  
  
  
* **特点**：这是一条被誉为“广东最美省道”的粤北人文与林海自驾经典公路。沿着S244省道自驾，穿行在新丰云髻山与始兴深渡水瑶族乡的竹海溪流间，清泉怪石，老树古桥；在始兴看满堂客家大围的宏伟防御工事，最终在南雄古街巷与梅关古道（梅开五岭）的青石板路上，追寻中原先民南迁岭南的千古足迹。
