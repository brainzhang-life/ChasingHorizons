### 经典旅游路线与自驾路线

#### 路线一：环千岛湖最美公路自驾线
* **特点**：这是一条名扬全国的浙江最美环湖自驾景观线。沿着获得最美公路封号的千汾公路与淳杨公路自驾，一侧是郁郁葱葱的江南丘陵，一侧是宛如内海般浩瀚、千百个翠绿岛屿点缀其间的千岛湖碧波；可在千岛湖大桥俯瞰全景，在桐庐富春江和瑶琳仙境的大型石钟乳溶洞内探秘，尽显江南山水的灵秀动人。
* **行车路线**：杭州市 → 富阳 → 建德 → 淳安千岛湖 → 桐庐（瑶琳仙境/富春江） → 临安大明山 → 返回杭州。
* **路线路段距离与地图**

#### 路线二：浙东山海与唐诗之路自驾线
* **特点**：这是一条沿着唐代诗人荒野泛舟吟游足迹、领略浙东名山与东海海岛的文化自驾线。从西子湖畔杭州出发，在绍兴鲁迅故里与兰亭前寻觅书圣墨香；在天台山国清寺前听千年隋梅的梵音；登临雁荡山和楠溪江，看飞瀑流泉与谢灵运笔下的奇峰叠峦；最后自驾跨海前往舟山普陀山，在海天佛国的潮音洞前，领略东海山海信仰的博大精深。
* **行车路线**：杭州 → 绍兴（鲁迅故里/兰亭） → 新昌（大佛寺） → 天台山（国清寺） → 神仙居 → 温州（雁荡山/楠溪江） → 台州临海古城 → 宁波天一阁 → 舟山普陀山。
* **路线路段距离与地图**

  | (3) 新昌 | (4) 天台山 | 240.0 公里 |
  | (4) 天台山 | (5) 神仙居 | 222.7 公里 |

  | (8) 宁波天一阁 | (9) 舟山普陀山 | 103.0 公里 |
