### 经典旅游路线与自驾路线

#### 路线一：青海湖环线经典自驾线
* **特点**：最受喜爱的的高原入门自驾线，看圣湖碧波、金黄油菜花与洁白盐湖。
* **行车路线**：西宁 → 塔尔寺 → 日月山 → 倒淌河 → 青海湖二郎剑 → 黑马河（赏圣湖日出） → 茶卡盐湖（天空之镜） → 刚察县 → 金银滩草原 → 祁连山草原 → 门源万亩油菜花海（7月） → 西宁。
