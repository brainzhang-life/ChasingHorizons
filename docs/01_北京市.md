### 经典旅游路线与自驾路线

#### 路线一：北京经典5天4夜探索之旅
* **适合人群**：首次来京的游客、亲子游、深度文化体验。
* **参考预算**：人均 1300 元左右。
* **行程规划**：
  * **Day 1**：天安门广场 → 毛主席纪念堂 → 故宫博物院 → 什刹海 → 南锣鼓巷（感受老北京的皇家威仪与胡同烟火）。
  * **Day 2**：八达岭长城 → 明十三陵 → 鸟巢 → 水立方（登雄关，观奥运地标）。
  * **Day 3**：孔庙国子监 → 雍和宫 → 五道营胡同 → 簋街 → 天坛（祈福古刹，品尝京城美食）。
  * **Day 4**：清华/北京大学（外观） → 颐和园 → 圆明园（夏）/ 香山公园（秋）（皇家园林与名校风采）。
  * **Day 5**：国家大剧院 → 中国国家博物馆 → 三里屯太古里（人文艺术与现代潮流的碰撞）。

#### 路线二：北京深度游10日经典路线
* **参考预算**：人均 2500 元左右。
* **行程规划**：
  * **Day 1**：天安门 → 毛主席纪念堂 → 人民大会堂 → 故宫 → 景山公园 → 王府井
  * **Day 2**：天坛 → 中国国家博物馆 → 国家大剧院
  * **Day 3**：北海公园 → 恭王府 → 后海
  * **Day 4**：什刹海 → 鼓楼 → 南锣鼓巷
  * **Day 5**：孔庙国子监 → 雍和宫 → 五道营胡同 → 簋街
  * **Day 6**：奥林匹克公园 → 颐和园
  * **Day 7**：清华大学/北京大学 → 圆明园
  * **Day 8**：八达岭长城 → 明十三陵
  * **Day 9**：北京798艺术区 → 三里屯太古里
  * **Day 10**：北京动物园 → 玉渊潭公园 → 中央广播电视塔

#### 路线三：北京古都文化自驾游（2日经典）
* **特点**：串联京城核心历史遗迹，适合周末自驾深度游。
* **行车路线**：天安门广场 → 北京故宫 → 南锣鼓巷 → 什刹海 → 恭王府 → 王府井大街 → 天坛公园 → 圆明园遗址公园 → 颐和园。

#### 路线四：延庆百里山水画廊自驾线（1-2日）
* **特点**：京郊最美避暑自驾路线，四季山水如画。
* **行车路线**：北京市中心 → 延庆区 → 百里山水画廊（沿途经过燕山天池、硅化木群、乌龙峡谷、滴水壶、朝阳寺、龙王庙）。

#### 路线五：京郊长城经典自驾游（2-3日）
* **特点**：穿越崇山峻岭，打卡不同风貌的长城雄姿。
* **行车路线**：北京市中心 → 居庸关长城 → 八达岭长城 → 慕田峪长城 → 黄花城水长城 → 古北水镇 → 司马台长城。
