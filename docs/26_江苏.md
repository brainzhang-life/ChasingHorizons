### 经典旅游路线与自驾路线

#### 路线一：环太湖水乡古镇生态游
* **特点**：烟波浩渺的太湖风光，小桥流水的千年古镇，典雅端庄的古典园林。
* **行车路线**：苏州园林 → 吴中太湖景区 → 同里古镇 → 周庄古镇 → 无锡太湖鼋头渚 → 灵山大佛 → 拈花湾 → 宜兴竹海 → 常州天目湖/环球恐龙城 → 南京夫子庙/中山陵 → 镇江三山 → 扬州瘦西湖 → 泰州凤城河 → 南通狼山 → 盐城湿地 → 连云港花果山 → 徐州云龙湖。
