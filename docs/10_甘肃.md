### 经典旅游路线与自驾路线

#### 路线一：大西北黄金大环线（青甘大环线甘肃段）
* **特点**：大西北自驾的首选，融戈壁、荒漠、丹霞、石窟人文于一体。
* **行车路线**：兰州 → 武威（雷台景区） → 张掖（七彩丹霞/平山湖大峡谷） → 酒泉 → 嘉峪关（天下第一雄关） → 玉门 → 瓜州（品尝甜瓜，观大地之子） → 敦煌（莫高窟/鸣沙山月牙泉） → 玉门关/雅丹魔鬼城 → （进入青海大柴旦/茶卡/西宁） → 返回兰州。

#### 路线二：河西走廊丝绸之路自驾线
* **特点**：重走张骞凿空西域与玄奘西行之路，体验雄浑边塞诗意。
* **行车路线**：兰州 → 武威 → 永昌 → 山丹军马场 → 张掖七彩丹霞 → 酒泉 → 嘉峪关 → 玉门 → 瓜州 → 敦煌莫高窟 → 鸣沙山月牙泉 → 阳关 → 玉门关。

#### 路线三：璀璨世界文化遗产研学游
* **特点**：甘肃拥有 7 处世界文化遗产，此路线专为深度历史与艺术爱好者设计。
* **行车路线**：天水（麦积山石窟） → 兰州 → 武威 → 张掖 → 酒泉 → 嘉峪关 → 玉门关 → 瓜州锁阳城遗址 → 敦煌莫高窟 → 敦煌悬泉置遗址。

#### 路线四：奇峡秀水黄河风情自驾线
* **特点**：追随母亲河的足迹，看黄河石林与高山峡谷相映成辉。
* **行车路线**：白音风情线（景泰黄河石林） → 兰州百里黄河风情线 → 永靖黄河三峡 → 炳灵寺石窟 → 玛曲黄河首曲。

#### 路线五：崆峒问道与红色圣迹自驾线
* **特点**：道教文化、红色精神与黄土高原大褶皱地貌的完美结合。
* **行车路线**：兰州 → 会宁（红军会师旧址） → 静宁 → 平凉（崆峒山） → 泾川（大云寺·王母宫） → 庆城（周祖陵） → 华池南梁红色景区。
