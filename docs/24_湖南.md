### 经典旅游路线与自驾路线

#### 路线一：湘西风情与山水秘境自驾线
* **特点**：领略奇峰三千的张家界地貌，沉浸于凤凰古城与苗族侗族风情。
* **行车路线**：长沙（橘子洲/岳麓山） → 湘潭韶山（毛泽东故居） → 衡阳南岳衡山 → 郴州东江湖（雾漫小东江） → 邵阳崀山 → 怀化洪江古商城 → 湘西凤凰古城 → 吉首矮寨大桥 → 芙蓉镇 → 张家界国家森林公园/天门山 → 常德桃花源 → 岳阳楼/洞庭湖 → 长沙。
