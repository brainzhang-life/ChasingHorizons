### 经典旅游路线与自驾路线

#### 路线一：闽东北山海画廊自驾线
* **特点**：饱览武夷大王峰秀色、闽南土楼奇观与东海黄金海岸。
* **行车路线** marriages: 福州（三坊七巷） → 宁德（太姥山/嵛山岛） → 南平武夷山 → 三明（泰宁大金湖/沙县） → 龙岩（古田会议旧址/永定土楼） → 漳州（东山岛/南靖土楼） → 厦门（鼓浪屿） → 泉州（开元寺/西街） → 莆田湄洲岛 → 福州。
